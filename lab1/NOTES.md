In Lab 1, I set up my development environment using WSL2 and installed PostgreSQL 15, connecting it with DBeaver for easier database management. I learned to use the PostgreSQL shell (psql) for running SQL queries, understanding the difference between Linux commands ($) and SQL commands (=#). I created a table called books_read and practiced basic SQL operations like SELECT, ORDER BY, LIMIT, GROUP BY, and aggregate functions such as COUNT(), SUM(), and AVG(). I also learned to format dates using TO_CHAR() and filter grouped results with HAVING.

For version control, I practiced using Git commands like git add, git commit, and checking status with git status. I also understood how directory structure affects Git and the importance of working in the correct folder.

During the lab, I faced some challenges, like accidentally running SQL queries outside PostgreSQL, causing errors, syntax mistakes with LIMIT, and unfinished commit messages in Git. I solved them by carefully checking the terminal type, writing complete SQL statements, and closing quotation marks in commit messages.

Key Points Learned

Set up PostgreSQL and connected it with DBeaver.

Difference between Linux terminal $ and PostgreSQL terminal =#.

Created and managed tables with different data types.

Used basic SQL commands: SELECT, ORDER BY, LIMIT.

Used aggregate functions and grouping: COUNT(), SUM(), AVG(), GROUP BY, HAVING.

Learned to format dates with TO_CHAR().

Git workflow: git add, git commit, git status, importance of correct paths.

Learned to identify errors and solve problems in SQL, Git, and terminal commands.

Overall: Lab 1 taught me the basics of SQL, database management, and Git workflow. I learned to organize files, write queries, and solve small errors, which strengthened my technical and problem-solving skills
